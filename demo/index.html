The Vicsek model

This demo simulates N self-propelled particles in a two-dimensional periodic box of side L = 5. Each particle has position \(\mathbf{r}_i(t)\) and heading \(\Theta_i(t)\). Particles move at speed v and tend to align with neighbors within interaction radius R, perturbed by noise strength \(\eta \in [0,1]\).

Update rules

At each time step \(\Delta t = 0.25\), the simulation applies two synchronous updates:

\[ \Theta_i(t+\Delta t) = \arg\!\left( \frac{1}{|s_i(t)|}\sum_{j\in s_i(t)} v\,e^{i\Theta_j(t)} + \eta\,e^{i\xi_i(t)} \right) \]
\[ \mathbf{r}_i(t+\Delta t) = \mathbf{r}_i(t) + v\,\Delta t\,(\cos\Theta_i(t),\,\sin\Theta_i(t)) \]

The neighbor set is \(s_i(t)=\{j : |\mathbf{r}_i(t)-\mathbf{r}_j(t)|minimum-image distances across periodic boundaries, and \(\xi_i(t)\sim\mathrm{Uniform}(-\pi,\pi)\). Positions advance using the current heading \(\Theta_i(t)\), not \(\Theta_i(t+\Delta t)\).

Order parameter

Instantaneous polarization measures global alignment of headings:

\[ P(t) = \left|\frac{1}{N}\sum_{i=1}^{N} e^{i\Theta_i(t)}\right| \]

\(P(t)\approx 1\) means strong flocking (aligned headings); \(P(t)\approx 0\) means disorder. The live plot and sparkline track \(P(t)\) as the simulation runs.

What you should see (Problem 1)

  • Low noise (\(\eta=0.1\)): after a short transient, \(P(t)\) rises toward 1 and particles form a coordinated flock.
  • High noise (\(\eta=0.9\)): \(P(t)\) stays low with fluctuating headings — a disordered state.
  • Periodic box: particles leaving one edge re-enter on the opposite side (Pac-Man wrap).

About this demo

Blue dots are particle positions; red segments show heading direction. Sliders let you explore how \(\eta\), \(R\), \(v\), and \(N\) affect flocking. The JavaScript engine mirrors the update rules in vicsek.py qualitatively; random number differences mean trajectories are not bit-identical to the Python runs.
